# Try.Wepp 피봇 아이디어: 스타트업 아이디어 트렌드 리포트 플랫폼

> 작성일: 2026-04-03

---

## 핵심 컨셉

모든 사용자가 자신이 만들려는 서비스 아이디어를 **익명으로 제출**하면,
플랫폼이 이를 AI로 분석해 **트렌드 리포트**를 전체 사용자에게 제공한다.

```
사용자 익명 제출 → AI 분석 및 클러스터링 → 트렌드 리포트 → 전체 공유
```

---

## 현재 제품 대비 개선점

| 문제 | 현재 Try.Wepp | 피봇 후 |
|---|---|---|
| Cold start | 빌더·리뷰어 모두 없으면 무가치 | 제출 10개만 있어도 트렌드 생성 가능 |
| 리텐션 | 런칭 후 재방문 이유 없음 | 리포트 업데이트마다 돌아올 이유 생김 |
| 크레딧 경제 | 500개 리뷰 필요 (수학적으로 깨짐) | 제출하면 바로 리포트 접근 |
| 가치 제안 | 3가지로 분산 (마켓플레이스·테스트툴·분석) | "아이디어 모아 트렌드 보여줌" 하나로 명확 |

## 강점

### 1. 데이터 플라이휠 (핵심 경쟁 우위)
```
더 많은 제출 → 더 정확한 트렌드 → 더 높은 가치 → 더 많은 제출
```
선순환이 돌기 시작하면 **독점 데이터 자산**이 형성되어 후발주자가 복사하기 어렵다.

### 2. 익명성이 마찰을 극적으로 낮춤
현재는 "내 프로젝트를 공개 등록"해야 해서 심리적 장벽이 높다.
익명 제출은 아직 아이디어 단계인 99%의 잠재 창업자를 끌어들일 수 있다.

### 3. 기존 경쟁사와 차별화된 포지셔닝
- CB Insights, Crunchbase: **이미 존재하는 회사** 기반 데이터 (후행 지표)
- Try.Wepp 피봇: **아직 만들어지지 않은 아이디어** 집계 (선행 지표)
- 시장에서 이 포지션을 가진 플랫폼이 없다.

### 4. 기존 코드 재활용 가능
현재 구현된 카테고리 시스템 (SaaS, Marketplace, Health 등), 분석 대시보드,
WTP 슬라이더가 그대로 활용 가능한 인프라다.

---

## 리스크

### 1. 역선택(Adverse Selection) 문제 — 가장 치명적
진짜 좋은 아이디어를 가진 사람일수록 익명이더라도 제출하지 않을 가능성이 높다.
"내 아이디어가 트렌드에 잡히면 누군가 먼저 만들어버릴 수도 있다"는 심리.

결과적으로 플랫폼에 쌓이는 건:
- 이미 널리 알려진 뻔한 아이디어
- 실현 가능성 없는 아이디어
- 스팸성 제출

**→ 리포트 품질 저하의 구조적 위험.**

**완화 방안**: 제출자에게 "이 아이디어 범주의 경쟁 강도·시장 공백 분석"을 개인 리포트로 돌려주는 것 — 제출에 대한 인센티브를 만들어 역선택 완화.

### 2. "분석"의 실체 문제
단순 카테고리 집계면 아무도 돈 안 낸다. 차별화된 인사이트를 위해 필요한 것:
- **클러스터링**: 유사 아이디어 묶기 및 포화도 측정
- **시계열 분석**: 어떤 카테고리가 증가/감소 중인지
- **지역·시장별 패턴**: 한국 vs 글로벌 비교
- **교차 분석**: 실제 투자 데이터와 매핑

LLM 요약 수준으로 시작하면 진입장벽이 없으므로, 분석 깊이가 핵심 역량이 된다.

### 3. 수익화 타이밍 딜레마
```
무료 트렌드 리포트 → 제출은 쌓임 → 아무도 안 냄
유료 트렌드 리포트 → 데이터 없어서 리포트 없음 → 아무도 안 냄
```
초반 6~12개월은 무료로 데이터를 쌓아야 하며, 이 기간의 생존 전략이 필요하다.

### 4. 리포트 신선도 유지
주 1회 트렌드 리포트가 의미 있으려면 매주 충분한 신규 제출이 필요하다.
DAU가 낮으면 리포트가 정체되어 재방문 이유가 사라진다.

## 유저 세그먼트 및 수익화

| 세그먼트 | 제출 동기 | 지불 의향 | 역할 |
|---|---|---|---|
| 예비 창업자 | "내 아이디어 얼마나 독창적?" | 낮음 | **공급망** |
| 현직 스타트업 팀 | 시장 검증, 경쟁 파악 | 중간 | **공급망 + 소비자** |
| VC / 액셀러레이터 | 딜플로우 조기 발굴 | **높음** | **핵심 고객** |
| 대기업 이노베이션팀 | 화이트스페이스 탐색 | **높음** | **핵심 고객** |

**수익화 구조 (권장)**:
- **Free tier**: 주간 트렌드 요약 리포트 (제출 시 접근)
- **Pro tier** ($49~99/mo): 카테고리별 심층 리포트, 시계열 데이터
- **Enterprise tier** (커스텀): VC/대기업용 raw 데이터 API, 맞춤 리포트

## 성공 조건

1. **익명 제출 품질 보증**: 카테고리 강제 선택, 최소 설명 길이, 중복 감지, 스팸 필터
2. **분석이 실제로 차별화된 인사이트**여야 함 — 클러스터링 + 시계열이 핵심
3. **B2B 수익화 경로**를 처음부터 설계 (VC 구독, 기업 리포트)
4. **데이터 축적 기간(6~12개월)의 생존 전략**: 콘텐츠 마케팅, 파트너십
5. **제출 인센티브 설계**: 제출하면 본인 아이디어에 대한 개인 분석 리포트 제공
